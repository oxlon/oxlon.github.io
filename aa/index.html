1 Motivation & scope

Where the Real-Time module (15.7.2) answers "how is the economy doing now?", Advanced Analytics answers the forward and structural questions: where are the headline indicators heading (predictive modelling), which drivers carry the signal (machine learning), and what hidden structure the Ministry's data holds (data mining). All three run on the same harmonised store and are validated against conventional benchmarks.

5 Methodology

Predictive (15.7.3.1)

For consistency, this presents the Ministry's canonical ensemble — the same validated, model-selected forecast as the flagship report (Module 11.1) — so the institution carries one GDP/inflation outlook rather than a different number each time. The ensemble already satisfies the TT: candidate models (AR1, Potential, ECM, VAR, BVAR) are scored by an expanding-window out-of-sample backtest, those that fail to beat a random walk are dropped, and the survivors are combined by inverse-RMSE² (precision) weights. Skill (ensemble RMSE ÷ random-walk RMSE) is below 1 for all three indicators; bands are the published p80.

Machine learning (15.7.3.2)

Supervised regression (gradient boosting, random forest, regularised linear) with time-series cross-validation to avoid look-ahead. Feature importance gives explainability; CV-drift defines the monitoring trigger.

Data mining (15.7.3.3)

Unsupervised: hierarchical (Ward) + k-means clustering on standardised regional profiles, PCA for visualisation, and pairwise correlation for association discovery.
